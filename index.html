How to connect your API key
1
Go to console.anthropic.com and sign in or create a free account.
2
Navigate to Settings → API Keys and click Create Key. Copy it immediately — Anthropic only shows it once.
3
Go to Settings → Billing and add a payment method. Load at least $5 in credits — the API won't work without a credit balance.
4
Click Connect API key in the top right, paste your key, and hit Save. You're ready to generate.
⚠ A few things to know: Each generation costs roughly $0.03–0.05 (3–5 cents). A $5 credit balance covers ~100–165 generations. Credits won't auto-renew unless you enable it. Your API key is stored only in your browser — never sent anywhere except Anthropic directly.
